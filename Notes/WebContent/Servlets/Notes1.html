Servlets

A program that runs on the web/application Server. (Server side program)

  • It is a middel layer between (request from web browser i.e client) and (database of server i.e server).
  • Servelts generates dynamic web pages.
  • Servlet is a web component that is deployed on the server to create a dynamic web page.

Before we servlets, We use CGI (Common Gateway Interface)

The Common Gateway Interface (CGI) provides the middleware between WWW servers and external databases and information sources.

Servlet API

The javax.servlet and javax.servlet.http packages represent interfaces and classes for servlet api.
The javax.servlet package contains many interfaces and classes that are used by the servlet or web container. These are not specific to any protocol.
The javax.servlet.http package contains interfaces and classes that are responsible for http requests only.
